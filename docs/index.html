    • 磅單抄寫
    • Google Maps 距離截圖

    CHAP-V

    • 油單/磅單留存
    • GPS形成軌跡
    • 串接原盤查系統

  • 數據儲存

    現況做法

    • Excel檔案
    • 電腦資料夾
    • 紙本檔案夾

    CHAP-V

    • 雲端資料庫保存
    • 搜尋、列印以利盤查需求

  • 碳盤查計算

    現況做法

    • 排放係數源多 → 查找困難
    • Excel公式計算來源多、拖拉易錯
    • 上下游運具資料取得凌亂不易、容易重複計算等問題

    CHAP-V

    • Excel化結果（ISO查證用）
    • 可視化軌跡趨勢（減碳分析用）
    • 串上、下游運具整合自動排除與計算

  • 減碳分析

    現況做法

    • 解析度低，資訊不足以分析如何減量

    CHAP-V

    非監督式AI資訊分析

    減碳因素分析（CFA）排碳因子警示
    • 解析度高 → 資訊多
    • 趨勢行為解耦
